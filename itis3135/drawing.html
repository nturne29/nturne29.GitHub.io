Interactive Drawing with p5.js

About This Sketch

This interactive drawing uses the p5.js library to create a bouncing ball animation. The ball changes direction when it hits the edges of the canvas.

How It Works

  • The ball starts at position (100, 100)
  • It moves with speed 2.5 horizontally and 2 vertically
  • When it hits any edge, it bounces back
  • The animation runs at 60 frames per second
