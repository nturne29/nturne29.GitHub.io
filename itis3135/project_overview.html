1. Project Overview

Purpose

A dynamic, modern portfolio website showcasing Maxwell Peterson's engineering projects, Eagle Scout achievements, and professional background. The site will emphasize interactive elements (fade-in animations, PDF viewers, and Timelines) for an engaging user experience.

Intended Users

  • Potential employers in engineering/tech.
  • Academic peers/professors.
  • STEM outreach organizations.

Content

  • Homepage: Professional headshot + project preview grid.
  • Project Pages: 8 dedicated pages (Jeep, CAD Carburetor, 3D Printing, etc.) with media galleries and embedded PDF reports.
  • About Page: Bio, resume, and Discovery Place Science role.
  • Eagle Scout Page: Leadership and project details.
